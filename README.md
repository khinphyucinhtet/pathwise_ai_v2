# PathWise AI

**PathWise AI: Personalized Career & Life Companion** is a front-end hackathon prototype designed to support students, graduates, and job seekers through interview preparation, resume building, career discovery, and mental wellbeing support.

This project was created by **Team Nextara** as a product demo concept for a hackathon setting. The current version is a **prototype / mockup-style interactive demo**, built to present the idea clearly and professionally while showing how the platform could work in a real product environment.

## Project Idea

Many young people experience stress, anxiety, and confusion during career preparation and job searching. They may struggle with:

- Lack of real interview practice
- Difficulty creating strong resumes
- Unclear career direction based on skills and interests
- Pressure from competition and expectations
- Limited support that combines career growth with wellbeing

**PathWise AI** was proposed as an AI-powered companion platform that reduces this stress while improving career readiness.

## Main Objectives

- Reduce stress in job seeking through guided support
- Provide interactive interview coaching with real-time feedback
- Help users build professional resumes
- Recommend suitable career paths based on user profile
- Encourage balanced career growth with wellbeing support

## Current Prototype Scope

This repository contains the **front-end demo prototype** of the idea using only:

- `HTML`
- `CSS`
- `JavaScript`

There is **no backend and no external API connection** in this version. All interactions are simulated with placeholder logic so the demo works offline in a browser.

## Key Features

### 1. AI Interview Coaching - Live Demo

The website includes a simulated interview video-call experience with:

- **HR AI** for behavioral and personality questions
- **Technical AI** for skill-based questions
- **Manager AI** for decision-making and real-world scenario questions

Interactive demo behavior includes:

- Selecting different interviewer roles
- Role-based mock questions
- Voice playback using browser speech synthesis
- Text or microphone-based answers
- Simulated feedback for:
  - Confidence
  - Clarity
  - Relevance
- End-of-session improvement report

### 2. AI Resume Builder

Users can fill in:

- Full name
- Target role
- Email and phone
- Education
- University / institution
- Skills
- Experience
- Projects
- Certifications
- Achievements

The site generates a live CV preview and includes a **print / PDF-style download simulation** for hackathon demonstration purposes.

### 3. Career Recommendation System

The career section collects user profile information such as:

- Skills
- Interests
- University
- Field of study
- Personality style
- Stress tolerance
- Preferred work style
- Preferred environment
- Leadership preference
- Communication strength
- Creativity level
- Comfort with data / numbers

Based on these inputs, the prototype recommends matching career roles and shows:

- Match percentages
- Short explanations
- Improvement tips

### 4. Wellbeing Support

The prototype includes a simple stress-level interaction where users select how stressed they feel. Based on this, the UI provides:

- Encouraging messages
- Short wellbeing tips
- Balanced, non-clinical support guidance

### 5. Dashboard & Progress Tracking

The dashboard presents demo metrics for:

- Interview score
- Resume completion
- Career match
- Wellbeing status

This helps communicate the platform as a more complete product concept rather than a single-feature tool.

### 6. Local Login / Sign Up Demo

The project also includes a basic local authentication-style UI using browser storage for demo purposes only.

This allows:

- Sign up
- Sign in
- Remember me
- Local account persistence in the browser

This is **not production authentication** and is included only to make the prototype feel more realistic.
